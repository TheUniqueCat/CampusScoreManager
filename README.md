# 学生成绩信息管理系统

## Summary
JavaWeb 课程设计作品。类似学校的教务管理系统，实现了其中的部分功能，包括学生的增删改查，成绩的增删改查等。

## Note

### 关于数据库
> * 数据库名称：smxy_class
> * 字符集：UTF-8
> * 先建立数据库，再运行.sql文件
> * 连接字符串在org.myweb.dao包中的Dao.java文件中

### 其他功能详细说明
